My Projects

Here are some of the projects I'm currently working on!

Cisco Sparkbot Tracker

I'm currently implementing a tracking system that allows for a Cisco Sparkbot administrator to extract and track commands that are entered for Cisco's Sales Associate Program sparkbot. Once I successfully implement and test this new functionality, I'm creating a website that can live-time stream the statistics and necessary information to a Sparkbot administrator's fingertips

Pseudomonous Psorter

Pesudomonas aeruginosa is a common type of bacteria that only cause minor problems for most people such as swimmer's ear. For the weak, however, this bacteria can cause a life-threatening infection. Similar to Klebsiella, I'm creating a web application that performs regression analysis on the potency of a hospital-acquired sample.

Klebsiella Classifier

Klebsiella pneumoniae are species of bacteria that are found in human noses, mouths, and gastrointestinal tracts, but are also one of the leading causes of hospital acquired pneumonia. My web application allows for nurses at a hospital to collect a sample of suspected klebsiella and upload it online, to which I run a program called Blast and perform regression analysis. The end result is a projected potency of the bacteria, crucial for hospitals to know so future bacterial infections can be contained.
